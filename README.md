# 📚 EduHub - Global Educational Platform

A modern, open-source educational platform where learners and educators worldwide can share knowledge and grow together.

## ✨ Features

- **📖 Comprehensive Resources** - Access thousands of educational materials
- **🌍 Global Community** - Connect with millions of learners
- **⬆️ Easy Upload** - Share any type of educational file
- **🎯 Organized Learning** - Well-structured categories
- **🔍 Advanced Search** - Find resources instantly
- **🌙 Dark Mode** - Toggle between light and dark themes
- **📱 Responsive** - Works on all devices

## 🚀 Supported File Types

- PDF Documents
- Videos (MP4, WebM)
- Presentations (PPT, PPTX)
- Documents (DOC, DOCX)
- Images (JPG, PNG)
- Code Files
- Spreadsheets
- eBooks (EPub)

## 🎨 Green Theme

The platform uses a professional green color scheme (#10b981) throughout for a calming and educational feel.

## 🛠️ Installation

1. Clone the repository
2. Open `index.html` in your browser
3. Or deploy to GitHub Pages, Netlify, or Vercel
